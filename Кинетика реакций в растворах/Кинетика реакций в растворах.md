- Многомерная регрессия для функции нескольких аргументов:

$$
\begin{array}{c}
\ln r_{0}(\mathrm{S_{2}O_{8}^{2-}})=\ln k+\alpha\ln C_{0}(\mathrm{S_{2}O_{8}^{2-}})+\beta\ln C_{0}(\mathrm{I}^{-}) \\
\downarrow \\
y = a + b \cdot x_1 + c \cdot x_2 \\
\downarrow \\
a = -5.0474\pm0.0978;\ b = 0.8963 \pm 0.0153;\ c=1.5562\pm0.0203\\
\downarrow \\
\boxed{k  = 0.00643 \pm 0.00063\ \frac{1}{\text{M} \cdot \text{c}};\ \alpha = 0.8963 \pm 0.0153;\ \beta= 1.5562\pm0.0203}
\end{array}
$$


```python
def func(x, a, b, c):

    return a + b * np.log(x[0]) + c * np.log(x[1])

C = np.array([C_m2s2o8, C_ki])
r0 = np.log(r0)

popt, pcov = curve_fit(func, C, r0)
coefs = unc.unumpy.uarray(popt, np.sqrt(np.diag(pcov)))

k = unc.umath.exp(coefs[0])
alpha = coefs[1]
beta = coefs[2]
```
