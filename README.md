# physha_practice

Сборник материалов по практикуму физической химии: отчеты, графики и экспорт в PDF. Формат — заметки Obsidian с перекрестными ссылками.

## Структура

### В корне
- `Практикум.md` — основной индекс/сводник со ссылками на разделы.
- `Практикум.pdf` — экспорт общего практикума в PDF.
- `practice-title.md` — титульный лист (LaTeX-блок).
- `Практикумы.base` — служебная база/метаданные.
- `Кинетика автоколебаний.pdf` — отдельный PDF по автоколебаниям.
- `кинетика_реакций_приближения.pdf` — отдельный PDF по приближениям.

### Папки
- `Кинетика реакций в растворах/` — графики эксперимента.
- `Исследование кинетики ферментативной реакции/` — графики (Lineweaver–Burk, Eadie–Hofstee, Woolf–Hanes).
- `Квазистационар и квазиравновесие/` — иллюстрации к случаям/приближениям.
- `Кинетика каталитических реакций/` — графики.
- `Кинетика_автоколебаний/` — расчеты/графики (Brusselator, Oregonator и др.).
- `Кинетика фотохимического разложения/` — графики.
- `Кинетика фотохимического разложения железа/` — графики по ферриоксалат-иону.
- `кондуктометрия/` — графики.
- `геткат/` — гетерогенный катализ: конспект, презентация, графики/данные.
- `organic/` — органический практикум: основной файл и заметки по отдельным веществам.
- `organic/attachemnt/` — изображения-вложения для `organic/`.
